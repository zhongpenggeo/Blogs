1. 列表list是用[ ]包住的以逗号分隔的数据集合

    所有对列表的解析均采用[ ]，不论是元素引用或取值

    [ ]表示空列表

2. 字典由键-值(key-value)对构成，一般可采用{ }表示

    取字典中对应键值，则采用 [ ]

    { }表示空字典

3. 函数调用中的参数传入 采用( )，元组可用 ( )

    ( ) 表示空元组
    
#### 索引
4. 数组一般用array([data])表示。索引用array[index1,index2]表示或者array[index1][index2]。
5. 对于list的索引，只能用list[index1][index2]，
