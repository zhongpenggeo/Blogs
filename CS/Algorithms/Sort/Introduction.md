## COmparison

| 算法     | 最差运行时间  | 平均时间      |
| -------- | ------------- | ------------- |
| 插入排序 | $$O(n^2)$$    | $$O(n^2)$$    |
| 归并排序 | $$O(nlog n)$$ | $$O(nlog n)$$ |
| 堆排序   | $$O(nlog n)$$ | -             |
| 快速排序 | $$O( n^2 )$$  | $$O(nlog n)$$ |
| 计数排序 | $$O(k+n)$$    | $$O(k+n)$$    |
| 基数排序 | $$O(d(n+k))$$ | $$O(d(n+k))$$ |
| 桶排序   | $$O(n^2)$$    | $$O(n)$$      |
