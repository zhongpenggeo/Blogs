### 小数如何用二进制表示

将该数字乘以2，取出整数部分作为二进制表示的第1位；然后再将小数部分乘以2，将得到的整数部分作为二进制表示的第2位；以此类推，直到小数部分为0。 

**特殊情况：** 小数部分出现循环，无法停止，则用有限的二进制位无法准确表示一个小数，这也是在编程语言中表示小数会出现误差的原因

以计算0.6为例：

> 0.6 * 2 = 1.2 ——————- 1 
> 0.2 * 2 = 0.4 ——————- 0 
> 0.4 * 2 = 0.8 ——————- 0 
> 0.8 * 2 = 1.6 ——————- 1 
> 0.6 * 2 = 1.2 ——————- 1 
> …………

### 转换为十进制

$$
0.6 = 1 * 2^{-1} + 0 * 2^{-2} + 0 * 2^{-3} + 1 * 2^{-4}
$$
