### 问题

把远程的结果进行数据可视化（绘制图表），但是远程服务器（如centos）没有GUI

一种是文件传回本地再进行可视化

另一种是远程可视化。这里针对第二种

Xshell（Xshell 5对于最新系统可能不行，因为ssh加密算法变了）

```shell
display *.png
```

第三种借助python

在需要打开图片的路径下输入：

```shell
python -m http.server 8888
```

然后再本地浏览器用服务器IP：端口的形式登录该目录，可以在浏览器中查看。
