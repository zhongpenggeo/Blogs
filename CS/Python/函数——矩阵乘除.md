## 2. 矩阵除法

**MATLAB:**

A / B == >  XA=B

A \ B ==>   AX=B

```MATLAB
% Evaluate mrdivide.
A / B 
% Evaluate mldivide.
A \ B
% Evaluate A B^{-1}.
A / B 
% Evaluate A^{-1} B.
A \ B
```

**Python:**

```Python
# Evaluate mrdivide.
np.linalg.lstsq(B.T, A.T)[0].T
# Evaluate mldivide.
np.linalg.lstsq(A, B)
# Evaluate A B^{-1}.
np.linalg.solve(B.T, A.T).T
# Evaluate A^{-1} B.
np.linalg.solve(A, B)
```



#### 2.1 matlab的左除

```
a\b
```

即ax=b，求x

在matlab中，如果A是m*n的矩阵，且m~=n，b是列向量，那么x=a\b是满秩或超秩矩阵ax=b的最小二乘解，即

x是(a*x-b)范数（norm）的最小值。



在numpy中，等价于`np.linalg.lstsq(a,b)[0]`

### solve和lstsq的不同之处

solve适用于满秩（fully determined）矩阵，此时计算更快

lstsq使用最小二乘法，在满秩矩阵时速度更慢。
